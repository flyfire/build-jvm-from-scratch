# Build JVM From Scratch

从零构建 JVM ，实验楼课程[《从理论到实战，手把手教你构建 JVM》](https://www.lanqiao.cn/courses/2849)对应源码仓库。

# 使用姿势

1. 可 clone 本仓库，切换到章节对应分支，查看相应章节的代码。
2. 可通过 github 的链接来查看代码和 diff。
3. 在 issue 里反馈问题和建议。
4. 联系作者，添加作者微信进一步沟通。
